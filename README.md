# Proyecto-satelite
Somos Germán Yassin y Elishka y en este repositorio presentamos nuestro trabajo sobre la implementación de un sistema satelital compuesto por un satélite y una estación de tierra.

El satélite capta datos del entorno (temperatura, humedad y distancia) y los envía a la estación de tierra, donde una interfaz gráfica en Python permite visualizarlos en tiempo real en gráficas.
